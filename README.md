# 💳 Spring Boot Payment Microservices

A scalable and secure payment processing system built using Spring Boot 3 microservices architecture. This project demonstrates real-world backend design with authentication, API Gateway, and containerized deployment.

---

## 🚀 Tech Stack
- Java 17
- Spring Boot 3
- Spring Security + JWT
- API Gateway
- PostgreSQL
- Docker & Docker Compose

---

## 🔥 Features
- Secure authentication using JWT
- Payment transaction service
- API Gateway for routing
- Microservices-based architecture
- Dockerized deployment

---

## 📁 Architecture
- Auth Service → Handles login & token generation
- Transaction Service → Processes payments
- API Gateway → Routes requests
